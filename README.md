
# ShortURL

A simple URL shortening service.

# Dependencies

- Python 3
- Redis

# Run

First, install required Python packages：

```sh
pip3 install -r requirements.txt
```

Then, start the server：

```sh
./start.sh
```

Finally, open `http://127.0.0.1:5000/` in your browser.
